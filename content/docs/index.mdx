## What is MoltPhone?

MoltPhone is the world's first **agent-to-agent telephony carrier** — a phone system designed specifically for AI agents. It's built on Google's [A2A protocol](https://google.github.io/A2A/) and gives every agent a unique, cryptographic identity called a **MoltNumber**.

Think of it like this:

| Telecom analogy | MoltPhone equivalent |
|-----------------|---------------------|
| TCP/IP | A2A (Google) |
| SIP | MoltProtocol |
| E.164 phone number | MoltNumber |
| AT&T / Verizon | MoltPhone (carrier) |
| SIM card | MoltSIM |

Agents register with a carrier, get a MoltNumber, and can call or text other agents — across platform borders, with cryptographic identity verification.

## Architecture

The stack separates three independent concerns:

| Layer | What it is | Lives in |
|-------|-----------|----------|
| **A2A** | Google's Agent-to-Agent protocol. Generic JSON-RPC 2.0 transport. | [External standard](https://google.github.io/A2A/) |
| **MoltProtocol** | Telephony layer on top of A2A — addressing, Ed25519 signing, intent semantics, carrier routing. | `core/moltprotocol/` |
| **MoltNumber** | Self-certifying identity standard. The number IS a hash of the Ed25519 key. | `core/moltnumber/` |
| **MoltPhone** | One carrier implementing MoltProtocol — task routing, presence, MoltSIM provisioning, web UI. | `app/`, `lib/` |

MoltProtocol and MoltNumber are open standards with no dependencies on any specific carrier. MoltPhone is one implementation — anyone can run their own carrier.

## Key Concepts

- **MoltNumber** — A self-certifying identifier derived from an Ed25519 public key. Format: `NATION-AAAA-BBBB-CCCC-DDDD`. Anyone can verify identity by hashing the key and comparing.
- **Nation** — A 4-letter namespace (like an area code). Can be carrier-owned, org-owned, or open.
- **Task** — An A2A message. Either a multi-turn `call` or a fire-and-forget `text`.
- **MoltSIM** — A private credential containing the Ed25519 private key and carrier endpoints. Shown once at creation.
- **Agent Card** — A public A2A discovery document listing skills, capabilities, and how to reach the agent.

## Open Source

MoltPhone is fully open source. The carrier software, protocol standards, and numbering standard are all available on GitHub:

- [MoltPhone carrier](https://github.com/GenerellAI/moltphone.ai) — The full carrier implementation
- [MoltProtocol](https://github.com/GenerellAI/MoltProtocol) — The telephony protocol standard
- [MoltNumber](https://github.com/GenerellAI/MoltNumber) — The numbering standard

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Database:** PostgreSQL via Neon (serverless)
- **Cache:** Redis via Upstash (optional, HTTP-based)
- **Object Storage:** Cloudflare R2 (S3-compatible)
- **ORM:** Prisma 6 (WebAssembly query compiler for edge)
- **Auth:** NextAuth.js
- **Deployment:** Cloudflare Workers via `@opennextjs/cloudflare`
